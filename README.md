# 🏎️ Shadow Performance - Sistema de Vendas Automotivas

Um sistema completo de e-commerce e gestão voltado para entusiastas automotivos. A plataforma oferece um catálogo robusto de peças de reposição, acessórios estéticos e componentes de alta performance (tuning), facilitando a busca por compatibilidade de veículos e a jornada de compra do usuário.

## 🚀 Funcionalidades

- **Catálogo Especializado:** Navegação por categorias (Mecânica, Estética, Performance, Som).
- **Filtro de Compatibilidade:** Sistema inteligente para filtrar peças exatas para projetos específicos (ex: *Honda Civic 2007*, *Fusca Motor a Ar*).
- **Carrinho e Checkout:** Fluxo de compra fluido com cálculo de frete e opções de pagamento.
- **Gestão de Estoque:** Painel administrativo para controle de entrada e saída de peças.
- **Wishlist de Projetos:** Os usuários podem salvar peças para futuras modificações em seus carros.
